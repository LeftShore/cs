差异化

  • 前缀规范

    通过统一的容易理解和书写的前缀来代替以前常用的css hack，来提高代码质量的一种方法。

IE 增强

目前只支持IE6,7;即将支持IE8; 部分功能处于试验阶段。

  • IE6
    • 选择器增强

      CSS3选择器更强大，这里增强点主要有：

      • 状态选择器

        用例说明：

        • 鼠标悬停(:hover)

          常用与实现菜单，富文本提示，悬停tab组建，等等...

        • 输入焦点(:focus)

          给输入型表单元素提供丰富的交互体验。
          原生支持从IE8开始，但IE7增强代码不与IE6共享。

        • 当前元素(:active)

          给输入型表单元素、连接等具备tabIndex属性的交互元素提供丰富的交互体验

        • UI状态(:checked,:disabled...)

          给输选择型表单元素提供丰富的交互体验
          原生支持从IE9开始，但IE7增强代码不与IE6共享。

        • Hash状态(:target,IE7 实现)
      • 结构选择器

        用例说明:

        • 首位元素(:first-child,:last-child)

          首位元素特殊边框等,背景差异等。常用与控制组建背景图形生成，解放以前模板中对首位行号的判断。

        • 单双行差异(:nth-child(odd),:nth-child(even))

          控制隔行样式差异(odd,even 只是特例，事实上这里可以是一个*n+k 表达式)

        • 其他(nth-child(?),nth-last-child(?),nth-of-type(?),nth-of-last-type(?))

          通过这种强大的位置表达式功能，解放模板中对行号的判断。

      • 属性选择器

        用例说明：

        • 分类样式

          如站内链接，链接类型等等差异化展现。

        • 其他

          增强后的代码可以动态支持CSS2的全部属性选择器功能，使用方式自己想象吧^_^

      • 选择器组合

        用例说明：

        • .classA.AndClassB

          IE6多class组合

        • .parent>.child

          直接子节点

        • .previous+.next

          相邻节点(选中next)

        • .previous~.next

          前置节点(只要存在更前的兄弟节点即可（不需要相邻），选中next)

        • .match:not(.unmatch)

          选中后排除

    • 其他功能类增强
      • 溢出控制(max/min-width/height)

        控制超出大小的切除和不足大小的补充

      • 固定定位(position:fixed)

      • PNG24 Alpha

        CSS 背景中运用 PNG24Alpha图像

  • IE7

      • 输入焦点(:focus)

        给输入型表单元素提供丰富的交互体验。
        代码不与IE6共享。

      • UI状态(:checked,:disabled...)

        给输选择型表单元素提供丰富的交互体验
        原生支持从IE9开始，但IE7增强代码不与IE6共享。

      • Hash状态(:target)

        给页内跳转(hash改变)增加样式支持

        原生支持从IE9开始，但IE7增强代码不与IE6共享。
